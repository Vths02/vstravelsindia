# VS Travels India — Website

Single-file static website (`index.html`) for VS Travels India — holiday packages, tour booking and Teerth Yatra (pilgrimage) trips.

## What's inside
- `index.html` — the entire website (HTML + CSS + JS), including all images embedded directly in the file. No external files or folders are required.

## Works everywhere
- Mobile, tablet, laptop, desktop — fully responsive layout.
- Works on all modern browsers: Chrome, Safari (iPhone/iPad/Mac), Edge, Firefox.
- No installation, build step, or server-side code needed — it's a plain static HTML file.

## How to host it

### Option A — GitHub Pages (free hosting)
1. Create a new GitHub repository (or use this one).
2. Upload `index.html` to the root of the repository.
3. Go to **Settings → Pages**.
4. Under "Build and deployment", set **Source: Deploy from a branch**, **Branch: main**, folder **/(root)**.
5. Save. GitHub will give you a live link like:
   `https://<your-username>.github.io/<repo-name>/`
   (takes 1–2 minutes to go live after first save.)

### Option B — Any other hosting
Just upload `index.html` to any static web host (Netlify, Vercel, cPanel, etc.) — no other setup required.

## Editing
Everything (styling, text, images, links) lives inside the single `index.html` file. Open it in any text/code editor to make changes.
